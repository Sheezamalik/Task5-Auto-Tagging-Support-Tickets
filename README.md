# Task 5: Auto Tagging Support Tickets Using LLM

**Objective:** Automatically tag support tickets into 
categories using a Large Language Model.

**Dataset:** Custom support tickets dataset (20 tickets)

**Approach:** 
- Zero-Shot Learning
- Few-Shot Learning
- Prompt Engineering

**Tags Used:** Technical Issue, Billing, Account, 
Connectivity, App Bug, Cancellation, Upgrade, 
Refund, Password, Data Usage

**Key Results:**
- Few-Shot learning outperforms Zero-Shot
- Each ticket gets top 3 most relevant tags
- Few-Shot gives more consistent results

**Tools:** Python, OpenRouter API, Pandas, 
Matplotlib, Seaborn

## How to Run
1. Open notebook in Google Colab
2. Add your OpenRouter API key in Cell 2
3. Run all cells
